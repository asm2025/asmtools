This needs the following dependencies:
Debian:
    sudo apt-get update
    sudo apt-get install -y build-essential libasound2-dev libgflags-dev libsnappy-dev zlib1g-dev libbz2-dev liblz4-dev libzstd-dev clang
Fedora:
    sudo dnf install alsa-lib-devel

Then do:
    git clone https://github.com/facebook/rocksdb.git
    cd rocksdb
    make shared_lib
    sudo make install
